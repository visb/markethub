# S2.8 — App cliente: vitrine, busca e carrinho

- **Fase:** 2
- **Epic:** Marketplace (cliente)
- **Status:** todo
- **Depende de:** [S1.6, S2.3]

## Objetivo
Telas do app cliente para navegar o catálogo, buscar e montar o carrinho multi-loja.

## User story
Como cliente, quero navegar produtos, buscar e adicionar ao carrinho, para montar minha compra (ref: `Merchant Home.jpg`, `Cart.jpg`).

## Critérios de aceite
- [ ] Home do merchant: stories/promoções, seções "Ofertas em destaque", "Mais comprados", "Recomendados" (S1.6 `sections`).
- [ ] Browse por categoria (Padaria, Hortifruti, Açougue, Bebidas, Mercearia) com lista paginada.
- [ ] Busca por produto/marca/departamento (S1.6 `search`).
- [ ] Card de produto com imagem, preço, promoção e botão adicionar; controle +/- de quantidade.
- [ ] Tela de carrinho ("MEU CARRINHO"): itens agrupados por loja, subtotais, variação de peso, remover/editar.
- [ ] Estados de loading/erro/vazio; consumo via `@markethub/api-client`.

## Escopo / Fora de escopo
Inclui: vitrine, busca, carrinho no app `customer`.
Não inclui: checkout/PIX/pedidos (S2.9).

## Notas técnicas
- Reusa `packages/ui` (tokens, Button, Screen) e navegação do shell (S0.6).
- Estado do carrinho sincronizado com a API (S2.3); otimista onde fizer sentido.
- Imagens do enriquecimento (Cosmos) já vêm do catálogo.
