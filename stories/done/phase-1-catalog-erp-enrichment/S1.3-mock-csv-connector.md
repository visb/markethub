- **Fase:** 1
- **Epic:** Integração ERP
- **Status:** done
- **Depende de:** [S1.2]

## Objetivo
Conector de referência baseado em CSV/JSON, para popular o catálogo sem depender de um ERP real.

## User story
Como dev, quero um conector mock, para desenvolver e testar todo o fluxo de catálogo ponta a ponta já agora.

## Critérios de aceite
- [ ] Implementação `CsvErpConnector` cumprindo a interface `ErpConnector`.
- [ ] Datasets de exemplo (produtos, preços, estoque) para 2+ supermercados fictícios coerentes com os screenshots.
- [ ] Comando/job que roda o sync completo via mock e popula `Product`/`Offer`/`Stock`.
- [ ] Suporta re-sync (delta): alterar preço/estoque no arquivo reflete no banco sem duplicar.
- [ ] Documentado como adicionar/editar dados de teste.

## Escopo / Fora de escopo
Inclui: conector concreto + fixtures + execução.
Não inclui: ERP comercial real (plugado depois pela mesma interface).

## Notas técnicas
- Arquivos em `apps/api` (ou `infra/fixtures`).
- Servir como contrato de exemplo para futuros conectores reais.
- Cobrir EANs válidos para exercitar a normalização da S1.5.
